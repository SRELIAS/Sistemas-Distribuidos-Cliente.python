Para executar o código fornecido, siga os seguintes passos:

    Abra dois terminais separados.

    No primeiro terminal, execute o código do servidor. Isso pode ser feito digitando o seguinte comando no terminal:


   Python server.py


    O servidor agora está em execução e aguardando por conexões de clientes. Aguarde até ver a mensagem de que o servidor está esperando por conexões.

    No segundo terminal, execute o código do cliente. Isso pode ser feito digitando o seguinte comando no terminal:


   Python client.py


    O cliente será executado e solicitará que você insira a mensagem a ser enviada para o servidor. Digite a mensagem desejada e pressione Enter.

    O cliente enviará a mensagem ao servidor e aguardará a resposta.

    Volte para o terminal do servidor e você verá a mensagem recebida do cliente sendo exibida no console do servidor.

    Após a exibição da mensagem recebida no servidor, o cliente será finalizado.

Certifique-se de que os endereços IP e portas definidos no código do servidor e do cliente estejam corretos e correspondam.

Ao seguir esses passos, você poderá testar a comunicação entre o cliente e o servidor, enviando mensagens do cliente para o servidor e recebendo as respostas correspondentes.
